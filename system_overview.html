📊 Survey Management System 기획서

1. 시스템 개요

Survey Management System은 교육 프로그램의 만족도 조사를 효율적으로 관리하고, 응답률을 높이기 위한 통합 플랫폼입니다. OpenAI를 활용한 설문 문항 자동 생성부터 응답 현황 분석, 리마인더 발송까지 전체 프로세스를 관리합니다.

2. 목적

주요 목적

  • 교육 만족도 조사 프로세스 자동화
  • 응답률 향상을 위한 효율적인 리마인더 시스템 구축
  • 데이터 기반 교육 품질 개선
  • 관리자의 업무 효율성 증대

3. 주요 기능

3.1 Survey 생성 및 관리

  • OpenAI 활용 설문 문항 자동 생성
  • Google Forms 연동
  • 설문 템플릿 관리
  • JSON 형식 다운로드 지원

3.2 대상자 관리

  • Google Sheets 연동을 통한 대상자 목록 관리
  • Excel/CSV 파일 업로드 지원
  • 대상자 그룹 관리
  • 실시간 데이터 동기화

3.3 응답 현황 분석

  • 실시간 응답률 모니터링
  • Plotly 기반 인터랙티브 차트
  • 만족도 분포 시각화
  • 부서별/그룹별 분석

3.4 리마인더 시스템

  • 미응답자 자동 식별
  • 맞춤형 리마인더 이메일 발송
  • 발송 현황 실시간 추적
  • Gmail API 활용 안정적인 메일 발송

4. 대상 사용자

사용자 유형 주요 활용 기능
교육 담당자
  • 설문 생성 및 관리
  • 응답 현황 모니터링
  • 리마인더 발송
교육 운영자
  • 대상자 목록 관리
  • 응답 데이터 분석
  • 결과 리포트 생성
관리자
  • 전체 시스템 관리
  • API 키 및 인증 관리
  • 사용자 권한 관리

5. 시스템 워크플로우

5.1 Survey 생성 프로세스

  1. 교육 대상, 목적, 요구사항 입력
  2. OpenAI를 통한 설문 문항 자동 생성
  3. 문항 검토 및 수정
  4. Google Forms 생성 또는 JSON 다운로드

5.2 응답 관리 프로세스

  1. 대상자 목록 등록 (Sheets/파일 업로드)
  2. 설문 링크 공유
  3. 실시간 응답 현황 모니터링
  4. 미응답자 대상 리마인더 발송
  5. 최종 응답 데이터 분석

6. 기대 효과

  • 설문 제작 시간 단축 (AI 활용)
  • 응답률 향상 (효율적인 리마인더 시스템)
  • 데이터 기반 의사결정 지원
  • 관리자 업무 효율성 증대
  • 교육 품질 개선을 위한 인사이트 도출

7. 기술 스택

구분 사용 기술
프론트엔드 Streamlit, Plotly
백엔드 Python
데이터 저장 Google Sheets
API 연동 OpenAI API, Gmail API, Google Sheets API
인증 OAuth 2.0
